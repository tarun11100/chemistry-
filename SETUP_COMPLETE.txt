💡 HOW IT WORKS

1. User opens login.html
2. User signs up with email, username, password
   → Data saved to localStorage
3. User logs in with credentials
   → Browser verifies against stored data
   → Session created (md_isLoggedIn = true)
   → Redirected to home.html
4. Home page checks if logged in
   → If yes: Show dashboard with username
   → If no: Redirect to login
5. User clicks Logout
   → Session cleared
   → Redirected to login
6. All data stored locally - NO SERVER NEEDED!
